# Image Enhancement and Restoration

This project is developed using Python, OpenCV, NumPy, and Streamlit to implement various image enhancement, filtering, and restoration techniques.

## Features

### Enhancement Techniques

* Negative Transformation
* Log Transformation
* Gamma Correction
* Contrast Stretching

### Filtering Techniques

* Mean Filter
* Median Filter
* Gaussian Filter
* Sharpening Filter

### Restoration Techniques

* Noise Removal
* Deblurring
* Wiener Filter

## Technologies Used

* Python
* OpenCV
* NumPy
* Streamlit
* SciPy

## How to Run

Install dependencies:

```bash
pip install -r requirement.txt
```

Run the project:

```bash
streamlit run app.py
```

## Input Formats

* JPG
* PNG
* JPEG

## Project Objective

To improve and restore digital image quality using image processing techniques through an interactive dashboard.
